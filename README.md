# Sentiment-Analysis

## How to run the project
1. Train the model by running the following command.
```
python train_model.py
```
2. Run the project.
```
python main.py
```
3. To test, Enter any comment and check the result.
4. To exit, enter one of the following codes
```
1. Exit
2. Stop
3. Escape
4. 0
5. Esc
```
